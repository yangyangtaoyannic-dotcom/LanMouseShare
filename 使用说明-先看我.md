# LanMouseShare 0.1.42 使用说明

这是一个局域网内两台 Windows 电脑共享鼠标、键盘、剪贴板和文件传输的小工具。

## 适合谁用

- 两台 Windows 电脑在同一个局域网。
- 想用一套鼠标键盘在两台电脑之间切换。
- 想在两台电脑之间拖文件发送。

## 安装前准备

- 两台电脑都需要安装。
- 建议 Windows 10 或 Windows 7 SP1。
- 需要 .NET Framework 4.8。
- 安装时需要管理员权限。

## 快速安装

1. 在电脑 A 上右键运行：

   ```text
   LanMouseShareSetup.exe
   ```

   选择“以管理员身份运行”。

2. 在电脑 B 上也运行同一个安装器。

3. 两台电脑都打开 LanMouseShare 窗口。

4. 等待自动发现对方电脑。

5. 如果没有自动发现，就在其中一台电脑里手动输入另一台电脑的 IPv4 地址。

6. 发起配对，并在另一台电脑上同意。

7. 设置屏幕方向：

   - 如果电脑 B 在电脑 A 右边，就在电脑 A 上选“对端在右”。
   - 如果电脑 B 在电脑 A 左边，就选“对端在左”。
   - 上下方向同理。

8. 把鼠标移动到对应屏幕边缘，就可以跨屏控制。

## 常用功能

### 鼠标跨屏

鼠标碰到你设置方向的屏幕边缘，并继续往外移动，就会切换到另一台电脑。

### 键盘输入

当前鼠标控制哪台电脑，两边键盘都应该输入到哪台电脑。

### 文件传输

把文件拖到 LanMouseShare 窗口里即可发送。对方电脑收到后会保存到桌面：

```text
Mouse传输文件
```

### 剪贴板

默认支持文字和图片剪贴板同步，可以在界面里关闭。

## 快捷键

| 功能 | 默认快捷键 |
| --- | --- |
| 暂停/恢复跨屏 | `Ctrl+F11` |
| 停止所有控制 | `Ctrl+F10` |
| 关闭程序 | `Ctrl+Alt+End` |

快捷键可以在软件界面里修改。

## 如果装过旧版本或新版本

如果安装时出现奇怪问题，比如：

- 打开旧安装器却跳到新版本。
- 服务启动失败。
- 配对状态异常。
- 鼠标键盘状态不对。

先运行：

```text
LanMouseShare-Rollback-Clean.bat
```

它会自动请求管理员权限，并清理旧服务、旧安装目录、旧自启动和旧配置。清理完之后，再重新运行 `LanMouseShareSetup.exe`。

## 注意事项

- 两台电脑必须装同一个版本，不要混用不同版本的 Agent 和 Service。
- 向日葵、Mouse without Borders、远程桌面、投屏软件这类工具可能会拦截模拟鼠标输入，不保证能被稳定控制。
- 如果鼠标状态异常，优先按 `Ctrl+F10` 停止控制。
- 如果仍然异常，退出两端软件后运行回退清理工具，再重新安装。
